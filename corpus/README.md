# OpenScientist Corpus

This directory contains first-party Markdown documents that agents can read
through the plane corpus interface.

## Documents

- `openscientist` - a sanitized platform guide for using OpenScientist across
  research, notes, skills, sandboxing, worktrees, steering, cloud runs, and deep
  runs.

## Agent Access

```bash
"$PLANE_TOOL_BIN" corpus list
"$PLANE_TOOL_BIN" corpus view openscientist
```
